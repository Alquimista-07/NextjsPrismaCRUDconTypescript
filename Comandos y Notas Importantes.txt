-----------------------------------------------------------------------------------------------------------
* El siguiente comando me permite crear una aplicación en Nextjs
-----------------------------------------------------------------------------------------------------------

	npx create-next-app nombre-app
	
  SETUP
  * Lo primero que se nos va a solictar es si queremos usar JS o Typescript (En este caso Typescript)
  * Lo siguiente si queremos usar ESLint (En este caso le damos Yes)
  * Posteriormente si queremos usar Tilewind (En este caso le damos Yes)
  * Si queremos usar la carpeta src (En este caso le damos Yes)
  * Si queremos usar el App Router (En este caso le damos Yes)
  * Y si queremos usar el import alias (En este caso le damos No)

-----------------------------------------------------------------------------------------------------------
* El siguiente comando me permite arrancar una aplicación en Nextjs
-----------------------------------------------------------------------------------------------------------

	npm run dev
	
-----------------------------------------------------------------------------------------------------------
* El siguiente comando me permite hacer el build de producción en Nextjs
-----------------------------------------------------------------------------------------------------------

	npm run build
